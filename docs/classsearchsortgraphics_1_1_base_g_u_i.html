searchsortgraphics.BaseGUI.BaseGUI ( )
inlinepackage

initializes the BaseGUI object and sets up all JPanels

Sets up a nonresizeable JFrame size BaseGUI.MAXX by BaseGUI.MAXY. Subpanels are also constructed. A welcoming background is presented in the rendering screen Screen.

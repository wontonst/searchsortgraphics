Ausführliche Beschreibung

advanced numbers array displayer.

The DynamicScreen class replaces the deprecated Screen class. Major improvements have been made. The DynamicScreen automatically adjusts bar size according to the size requested by the user during construction. Spacing, vertical scaling, and width are all dynamically generated. Furthermore, significant performance advancements have been made to displaying color. There are now only two arraylists that holds any type of color, versus the old system of having an arraylist for each color.

RoyZheng
